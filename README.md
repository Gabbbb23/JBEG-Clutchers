# Instructions

In your terminal run:
```bash
py -m venv .venv
```

then, run:
```bash
.venv/Scripts/activate
```

go to:
```bash
cd backend
```

Install packages:
```bash
pip install -r requirements.txt
```

Go back to previous repository
```bash
cd ..
```

go to:
```bash
cd frontend
```

```bash
npm install
```
